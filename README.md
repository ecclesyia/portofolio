# Portfolio

This is my personal portfolio website. It shows my projects, skills, background, and contact links in one page.

## What It Uses

- [main.html](main.html) for the page content and layout
- [style.css](style.css) for all visual styling
- [main.js](main.js) for interactivity, project cards, and page behavior

## How To Use

Open [main.html](main.html) in a browser to view the site locally.

If you want to edit the site, update the matching file for the part you want to change:

- Content and sections: [main.html](main.html)
- Colors, spacing, fonts, and layout: [style.css](style.css)
- Buttons, filters, modal behavior, and dynamic content: [main.js](main.js)
