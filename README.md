## 📖 About the Project

This website serves as my digital business card and a live archive of my work. Rather than using heavy frameworks, I chose to build this using **Vanilla HTML, CSS, and JavaScript** to keep the site extremely fast, lightweight, and responsive across all devices.

### Key Goals:
- **Project Archiving:** A showcase for past and current creations.
- **Progress Tracking:** Documenting milestones and skills as I learn.
- **Clean Performance:** Optimizing code for speed and simplicity.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic markup for structured, accessible content.
- **CSS3:** Responsive grid and flex layouts, custom animations, and clean styling.
- **JavaScript (ES6):** Dynamic interactivity and project rendering logic.
